-- Projet Perso : FireFit -- 

21.01.2026 :
Développer en C#, ce programme permets : 
    - D'entrée une intervention,
    - Son stockage dans un fichier .json,
    - Visualisation des interventions,
    - Filtrage par engin,
    
L'application est mené à évoluer, notamment avec le stockage des séances de sport.


25.01.2026 : 
Nouvelles fonctionnalités : 
    - L'entrée des séances de sport est maintenant disponible, également son stockage,
    - Possibilités de voir les statistiques d'interventions (le nombre pour chaque engin),
    - Possibilités de voir le nombre d'interventions pour chaque mois,
    - Possibilités de réinitialiser les données (interventions + sport),


Base prête pour l'évolution future du projet (v2).
